Module outline

Conditioner does not force you to write your modules in a certain way. Only if you want to unload your modules or pass custom options does it expect certain properties an methods to be there.

The following two basic module examples both contain an unload method and an options property. If your module will never be unloaded you can omit the unload method, the same goes for the options property.

JavaScript API
