Passing options to a module data-options

The data-options attribute allows you to set specific options to be passed to the Module when it's loaded.

{% highlight html %} ... {% endhighlight %}

Specifying generic options for mulitple modules of a certain type:

{% highlight javascript %}conditioner.setOptions({ 'modules':{ 'ui/Map':{ 'options':{ 'zoom':5 } } } });{% endhighlight %}

ConditionerJS will merge page level options with node level options and pass the resulting options to the activated module.

In the (not so exciting) example above the resulting option object would be {"zoom":10}, since node level options take precedence over page level options zoom level 10 overwrites zoom level 5.
