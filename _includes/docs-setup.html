Setup

The setup of Conditioner is a two step process.

As Conditioner needs RequireJS to function, download RequireJS at http://requirejs.org/, if you haven't played around with RequireJS before, follow the Getting Started guide for a quick meet and greet.

Now you've got a feel for RequireJS it is time to setup Conditioner.

RequireJS needs to know where to find the Conditioner framework. You'll need to add a path entry to the RequireJS configuration object pointing to the location of Conditioner. In the example below the file 'conditioner.js' is located in the 'lib' folder within the generic 'js' folder.

{% highlight javascript %} requirejs.config({ baseUrl:'js/', paths:{ conditioner:'lib/conditioner' } });{% endhighlight %}

Now that's setup, you can request a conditioner reference and have it start loading modules.

{% highlight javascript %}requirejs(['conditioner'],function(conditioner) { conditioner.loadModules(document); });{% endhighlight %}

Since we've not yet configured our modules it will not load anything for now.
